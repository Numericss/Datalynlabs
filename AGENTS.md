# AGENTS.md

## Cursor Cloud specific instructions

This is a **Next.js 16 static landing page** (no backend, no database, no external APIs).

### Quick reference

| Task | Command |
|------|---------|
| Install deps | `npm install` |
| Dev server | `npm run dev` (port 3000) |
| Lint | `npm run lint` |
| Build | `npm run build` |

- See `README.md` for section layout and booking CTA config.
- There are no automated tests in this project; validation is done via lint + build + manual browser testing.
- The only service needed is the Next.js dev server on port 3000.
